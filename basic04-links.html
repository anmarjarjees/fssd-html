Absolute Links and Relative Links

HTML

CSS

We have two major types of links, Absolute links and Relative links. Absolute link is for linking to other website while relative link is for linking to other pages on the same website. You can also use the anchor element to link to sections on the same page.

Anchor Element (inline element) HTML links are defined by an anchor tag a we place the word(s) in between a /a to make it a link (hyperlink) href attribute (mandotery): When linking to another document, we need to tell the anchor which document should be opened. The document's source is the purpose of the href attribute. Absolute Link => the full URL => https//www.example.com target attribute (optional): target="_blank
